# RoadWatch AI

RoadWatch AI is an intelligent road damage monitoring and complaint management platform.

## Features

- Interactive OpenStreetMap integration
- Real-time road complaint markers
- Complaint submission workflow
- Image upload support
- PostgreSQL backend integration
- Flask REST APIs
- Geospatial complaint tracking
- AI-ready architecture for pothole/crack detection

## Tech Stack

Frontend:
- React
- React Leaflet
- OpenStreetMap

Backend:
- Flask
- Flask SQLAlchemy
- PostgreSQL
